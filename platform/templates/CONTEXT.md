# CONTEXT — platform/templates/

The lesson skeletons each draft fills. Structure only; the voice comes from
`platform/conventions/STYLE.md`.

## What's here

- `concept-lesson.md` — the skeleton for a **concept/decision** lesson (teach one idea, name the
  trade-off, land the seam line).
- `build-lesson.md` — the skeleton for a **build** lesson (the reader builds the thing; the exercise
  extends the module throughline).

## Load / don't-load

- **Load:** the one template matching the lesson type, alongside `STYLE.md` and `AUTHORING.md`.
- **Do NOT load:** book internals or the vault here.

## Handoff

Used by `platform/pipeline/` AUTHOR. No gates own this folder.
