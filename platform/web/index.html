The Engine

Built by a System, Not Typed Out by Hand

The hard part was never the writing; it was building something that could write at this scale without turning into AI slop. So the real project is the engine: a production pipeline where the folder tree is a routing system, a written contract governs the prose, and nothing ships until it passes a gate. The books are what it produces. This is why they hold up.

A Routing System

Every folder carries a contract that tells an arriving AI agent what to load, what to skip, and where to hand off. The tree navigates itself, and a linter fails the build if a route breaks.

A Writing Contract

A single style spec governs voice across all seven books: one consistent tone, no AI tells, enforced mechanically. If a draft and the contract disagree, the contract wins.

Human Gates

The system does the routine work on its own and stops at named decision points that need a person: what to name a book, what to publish. Autonomy with a hand on the wheel.

Green Gates

Every code artifact runs behind a passing test before it ships, and every technical claim is grounded in a live source citation. Nothing reaches the page unverified.
