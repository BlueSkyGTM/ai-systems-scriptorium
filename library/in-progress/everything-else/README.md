# Everything Else (working title)

The antilibrary, written up. *Sans Python* held material it knew about but chose not to teach —
recorded on its [antilibrary page](../../completed/sans-python/src/antilibrary.md). This book turns
that held material into its own collection: the "Avec Python" candidates and the deliberately-cut
threads, each given the chapter the first book denied it.

**Thesis:** discernment is zero-risk. Knowing a thing exists and roughly where to find it is its own
skill; this book is where the "roughly located" becomes "located."

**Status:** in progress, seeded. The real title is set at `GATE-NAME-BOOK`.

**Seed:** [`antilibrary.md`](../../completed/sans-python/src/antilibrary.md) — the per-module cut list
and the Avec Python candidates.
