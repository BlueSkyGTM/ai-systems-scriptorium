# Systems Design Log

Copy this file to `systems-design-log.md` and complete the single entry below across the four
Module 5 exercises. Each exercise fills in specific fields; by the end of the fourth exercise
the entire entry is filled in and free of placeholder text. Do not fill in all fields at once
-- work through the exercises in order and add to the entry as you go.

Run `python check_prep.py --module 5` from `exercises/prep/` when all four exercises are done.
