# Module 8: The Hiring Loop End-to-End

An interview loop is not one test. It is a sequence of rounds, each run by a different person evaluating a
different thing: a recruiter checking that you can say what you do, a hiring manager probing how you
operate, an engineer pulling on a system design, a panel mixing all of it. You have built a piece of
preparation for each of those rounds across this book. This module sequences them and grades the whole.

The dossier you have assembled is the preparation: the decompositions, the full Algorithm runs, the
behavioral bank, the systems-design log, the portfolio-narrative document, the deliberate-practice reps.
Each module built one piece and a tool to check it. The capstone closes the loop by composing all of it
into a single graded verdict, the way the loop itself will grade you: stage by stage, with no partial
credit for the rounds you skipped.

## The Capstone Is a Grader

Consistent with everything this book argues, the capstone is not an essay about feeling prepared. It is a
grader. The grader composes the tools you already have, the completeness checker and the calibration
scorer, and maps your dossier onto the five interview-loop stages. It marks each stage ready or not, names
the gap when there is one, and returns a single readiness verdict. You do not decide you are ready; you
run the grader and it tells you, against the same evidence the loop will use.

This is the point the whole book has been building toward. The framework made your reasoning transferable.
The example banks loaded it into reflex. The portfolio module turned your work into a narrative. The
practice module calibrated you. The grader confirms the pieces hold together as one preparation, not five
disconnected efforts.

## The Five Stages

The loop this module grades has five stages: the recruiter screen, the hiring-manager round, the
systems-design round, the portfolio deep-dive, and the panel. Each evaluates something different, and each
draws on a specific part of your dossier. One stage, the technical coding screen, is named but not graded
here: its preparation is sourced separately, and the module is honest about that gap rather than pretending
the dossier covers it.

## The Four Lessons

1. **Sequencing the Preparation** maps each dossier piece to where it pays off and tells you what to build
   first when time is short.
2. **The Stages of the Loop** walks the five stages: what each evaluates and how to prepare it from the
   dossier you already have.
3. **Grading Your Dossier** runs the capstone grader, reads its readiness report, and closes the gaps it
   names.
4. **Closing the Loop** covers working the loop on the day and after: the readiness verdict, the offer
   stage, and the end of the book.

## What This Module Closes

By the end you have run the grader and driven it to a ready verdict, which means every stage of the loop is
prepared with evidence behind it, not a feeling. That is the whole claim of this book made concrete: you
walk in with a graded dossier and a method that holds under a question you have never seen. The
preparation is finished when the grader says it is, and then the only thing left is the room.
