# tools/

Keepers. Runnable, documented, meant for real use — the higher-bar tier of the lab
([`../CONTEXT.md`](../CONTEXT.md)). Each tool is its own folder, scaffolded from
[`../_template/README.md`](../_template/README.md), with a populated `## Connections` section.

A tool can **graduate** to its own repo or into `library/` — a deliberate move gated at `GATE-PUBLISH`
(see [`../../platform/HUMAN-GATES.md`](../../platform/HUMAN-GATES.md)).
