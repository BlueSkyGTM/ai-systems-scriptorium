# production/

Proven enhancers. Runnable, documented, meant for real use — the higher-bar tier of the lab
([`../CONTEXT.md`](../CONTEXT.md)). An enhancer earns its way here from `experiments/` only after it
works. Each one is its own folder, scaffolded from [`../_template/README.md`](../_template/README.md),
with a populated `## Connections` section.

A production enhancer **graduates** to a permanent home — injected into a book, or its own repo at
`GATE-PUBLISH` (see [`../../platform/HUMAN-GATES.md`](../../platform/HUMAN-GATES.md)). The first resident
will be **Open Brain** (spec in [`../../progression/open-brain.md`](../../progression/open-brain.md)).
